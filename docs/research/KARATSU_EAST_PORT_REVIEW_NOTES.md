# 唐津東港パイロット レビュー記録

## 初回セルフレビュー

- 民間釣りサイト由来の文章・魚種一覧・座標を使用していない
- 港湾施設の存在と一般の立入・釣り可否を分離した
- 複数の公式水深値を、根拠なく単一の釣り場水深へ変換していない
- 港湾照明・航路標識灯を常夜灯と判定していない
- 駐車場の存在と釣り目的の利用可否を分離した
- 規制情報が見つからないことを「禁止なし」としていない
- 地点単位の公的魚種情報を確認できないため、魚種配列を空にした
- 代表座標は具体的な釣り位置ではないことを明記した
