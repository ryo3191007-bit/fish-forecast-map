# 魚種生態調査 source 方針

優先順位は、国・自治体・水産試験研究機関・大学・公的研究機関、査読論文・研究報告・公的DB、水族館・博物館・専門団体、その他二次sourceの順です。

## 記録必須項目

sourceごとに、発行主体、発行・更新日、確認日、地域範囲、直接性、`supporting` / `checked` / `contradicting`、派生元を示す`derivationGroup`、直接支えるSchema pathを記録します。同一原典の転載・派生ページは独立根拠として重複カウントしません。

## 禁止・制限

- 漁獲量・水揚げ量・養殖・流通情報を遊漁の釣れやすさへそのまま転用しない。
- 産卵期を釣期と同一視しない。
- 一般的な生息水深を岸から釣れる水深と同一視しない。
- 民間釣りサイト、SNS、動画、ブログ1件だけで`medium`以上にしない。
- 本文、画像、地点DB、魚種DBを転載しない。
- スクレイピング、自動巡回、定期取得、高頻度アクセスを行わない。
