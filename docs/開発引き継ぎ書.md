## 2. 最重要の作業ルール

### 2.1 基本フロー

1. GitHub正本を確認する。
2. 次の作業を一つだけ整理する。
3. 1 Issue = 1 Draft PRとする。
4. 実装担当を明示する。
5. 差分、CI、Vercel、制約違反、事実根拠をレビューする。
6. 実ブラウザ確認が必要な場合はPreviewで確認する。
7. Draft解除・マージは利用者が判断する。
8. マージ確認後に限り、リンクされたIssueを`completed`として閉じる。

### 2.2 禁止事項

- 利用者の明示判断なしにPRをマージしない。
- 複数の機能作業を同じIssue・PRへ混在させない。
- 外部情報、利用条件、公式回答、実行主体を推測で確定しない。
- CI greenだけでマージ可と判断しない。
- 実施していないブラウザ確認、性能計測、外部送信、DB反映を完了扱いしない。
- Codex Cloud内のcommitや`make_pr`メタデータをGitHub反映済みと扱わない。
- 利用者本人が作成したPRをCodex作成と記録しない。
- 即時確認できる結果を、定期確認だけに任せて先送りしない。
- 不明な値をAI推測で`confirmed`にしない。
- placeholder URLや架空sourceを正式な根拠として保存しない。

### 2.3 実行主体の記録

branch、commit、PRを誰が作成したかを分離して記録します。

- Codex Cloud: 隔離環境内の編集、テスト、commit候補。
- ChatGPTのGitHub接続: GitHub API経由で実際に行ったbranch、commit、PR操作。
- 利用者: 利用者本人が手動で行ったpush、PR作成、Draft解除、merge等。
- GitHub上で確認できない主体・経路: `未確認`。

GitHub上のbranch、HEAD SHA、PR番号、Draft状態を取得するまで、GitHub反映完了と報告しません。

### 6.2 水深・3D地形

現在のsource:

- 第一source: `GEBCO_2026 Grid 15 arc-second`
- データ由来: `GEBCO_2026 TID Grid`
- fallback: `NOAA NCEI ETOPO 2022 60 Arc-Second Bedrock`

主な機能:

- 2D水深色分け
- 等深線とズーム別密度調整
- 陰影profile
- 陰影・等深線toggle

- 視点preset
- タップ地点の緯度経度、表示source、参考水深
- GEBCO失敗時のETOPO fallback
- 両方失敗時の通常地図fallback

水深・3D地形は参考表示です。港内、岩礁、根、瀬、正確な海岸線を保証せず、航海・安全判断には使用できません。

### 6.3 Weather / Marine

Weather API:

- 気温
- 天気コード
- 降水量・降水確率
- 風速・風向・突風

Marine API:

- 海面水温
- 潮位参考値
- 波高・波向・波周期
- 海流速度・海流方向

条件:

- `timezone=Asia/Tokyo`
- `forecast_days=7`
- Weather / Marineを独立取得
- 片方成功時はpartial表示
- 30分以内のcacheはfresh
- 24時間以内のcacheはAPI失敗時fallback
- 24時間超は使用しない

環境値は説明可能なSCORE v2へ本格反映していません。

## 8. Supabase / DBの状態

master data:

- `fish_species`
- `fishing_spots`
- `source_registry`

未設定、取得エラー、空データ時は静的データへfallbackします。

本人釣果:

- メールOTP / magic linkログイン
- 未ログイン時はlocalStorage
- ログイン済みかつDB利用可能時はSupabaseへ保存
- DB失敗時はlocalStorage fallback
- owner-scoped
- 削除は論理削除
- 旧localStorage記録は自動移行せず、選択した記録だけ明示移行

新しいDB変更は`supabase/migrations/`を正本とします。旧SQLファイルをmigration代わりに使用しません。

GitHub、Issue、PR、チャット、ログへ次を貼りません。

- Supabase `service_role` key
- DB password / connection string
- `.env.local`
- 実トークン、実パスワード、管理者権限値
- 海しるAPIキー等の認証情報
