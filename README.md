# fish-forecast-map

釣果予測マップは、福岡県糸島市西岸から唐津湾、伊万里湾、平戸方面までの陸っぱり釣り情報を地図で確認するための個人利用Webアプリです。堤防・磯場などでルアー釣り／エサ釣りを検討するときに、モック釣果、魚種、エリア、釣り方、釣れそう度スコアを一覧できます。

- 公開URL: https://fish-forecast-map.vercel.app/#map
- 利用目的: 個人利用向けのMVP検証
- データ: MVP v0.1ではモックデータのみ

## MVP v0.1 でできること

- MapLibre GL JSの地図でモック釣果地点マーカーを確認できます。
- 釣果カードで日付、エリア、場所、魚種、釣果数、サイズ、釣り方、出典を確認できます。
- 魚種フィルタ、エリアフィルタ、キーワード検索で絞り込みできます。
- 釣れそう度が高い順、日付が新しい順、日付が古い順で並び替えできます。
- 条件リセットボタンで、魚種 `all`、エリア `all`、キーワード空、並び順 `scoreDesc` に戻せます。
- 0〜100点の説明可能な簡易ルールベースの釣れそう度スコアと根拠を確認できます。
- ダーク／ネオン／ガラス風のダッシュボードUIでPC・スマホから確認できます。

## MVP v0.1 ではやらないこと

- 釣果登録フォーム、ユーザー自身の釣果登録。
- 外部釣果サイトの自動取り込み、記事URL貼り付け、検索履歴、オートコンプリート。
- Supabase/DB連携、ログイン/認証。
- 潮汐・天気・水温データ連携。
- 3D海底地形表示。
- 実績ベースの高度な予測、複雑な機械学習。
- ページネーション、URLクエリ同期。
- 有料API、有料地図サービス、有料ホスティングの追加導入。

## 使用技術

- フロントエンド: Next.js + TypeScript
- UI: CSS（`src/app/globals.css`）
- 地図: MapLibre GL JS
- データ: ローカルのモック釣果データ
- 品質確認: ESLint、TypeScript、Next.js build

## ローカル起動方法

依存関係は `package-lock.json` に固定しているため、通常は `npm ci` を使います。

```bash
npm ci
npm run dev
```

`npm run dev` を実行したら、ブラウザで `http://localhost:3000` を開きます。

## 品質確認コマンド

```bash
npm run lint
npm run typecheck
npm run build
```

## データ方針

MVP v0.1では外部サイトの取り込みを行わず、モックデータのみを使用します。釣果情報には出典名と出典URLのフィールドを持たせますが、第三者サイト本文の転載や自動収集は行いません。

将来的に外部情報を扱う場合は、利用規約、robots.txt、著作権、アクセス負荷を確認し、公式API、RSS、許可を得た情報源、ユーザー提供情報を優先します。アプリ上の釣れそう度は参考情報であり、実際の釣果を保証するものではありません。
